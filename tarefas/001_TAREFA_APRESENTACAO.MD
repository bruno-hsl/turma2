# 📘 Tarefa 01 – Posicionamento Profissional e Fundamentos Técnicos  
## 🚀 Turma 2 – Início da Jornada Dev Jr

---

## 🎯 Objetivo

Dar início à sua jornada como Desenvolvedor(a) Júnior de forma estruturada, trabalhando três pilares fundamentais:

- 🧠 Posicionamento profissional  
- 🛠️ Fundamentos técnicos  
- 📂 Organização com Git e GitHub  

Nesta primeira etapa da **Turma 2**, o foco é desenvolver mentalidade, disciplina e base técnica sólida.  
As tecnologias mais avançadas (Java, Spring Boot, Docker, Kafka, Kubernetes, AWS) serão trabalhadas nas próximas fases.

Aqui você começa aprendendo a se apresentar como profissional e a organizar sua evolução.

---

## 📅 Prazo sugerido

**5 a 7 dias**

Organize seu tempo. Disciplina faz parte do processo.

# ✅ Entregas Obrigatórias

---

## 1️⃣ Clonar o repositório base

Acesse o repositório oficial da Turma 2:

👉 `https://github.com/fernandoericofilho/turma2.git`

Clone o projeto:

```bash
git clone https://github.com/fernandoericofilho/turma2.git
```

Acesse a pasta de alunos:

```
cd turma2/alunos

mkdir seu_nome_sobrenome

cd seu_nome_sobrenome
```

⚠️ Use seu nome em letras minúsculas e separado por underscore. 

## 2️⃣ Criar arquivo de apresentação profissional

Crie um arquivo chamado:

```
tarefa-01.md
```

Dentro dele, escreva sua apresentação profissional (até 8 linhas) contendo:

```
Nome, cidade e formação (ou transição de carreira)
Tecnologias que está estudando
Seu objetivo profissional (ex: Backend com Java, Cloud, DevOps)
O que você está fazendo hoje para evoluir
Uma característica comportamental forte (ex: disciplina, organização, foco)
```
💡 Essa apresentação deve poder ser usada:

```
No LinkedIn
Em entrevistas
Em dinâmicas
Em networking
```
Evite texto genérico. Seja claro, direto e profissional.

## 3️⃣ Prática com Git (Fundamental)

Estude e pratique os seguintes comandos:

```
git status
git pull
git add
git commit -m
git push
```
Depois de criar seu arquivo:

```
git add .
git commit -m "feat: adiciona tarefa 01 - apresentação profissional"
git push origin main
```

📌 Atenção à mensagem de commit. Ela deve ser clara e padronizada.

## 4️⃣ Atualização Estratégica do LinkedIn
 
Atualize ou crie seu perfil com:

```
Foto profissional simples e bem iluminada
Título estratégico (ex: Desenvolvedor Backend em Formação | Java & Spring Boot)
Texto da sua apresentação na seção "Sobre"
Inclusão de cursos, bootcamps e projetos
```

Evite:

```
Perfil vazio
Texto genérico
Informações desorganizadas
```

Seu LinkedIn começa a ser sua vitrine profissional.

## 🎯 O que será avaliado

```
Clareza na apresentação pessoal
Organização do repositório
Uso correto do Git
Qualidade do resumo técnico
Capacidade de explicar conceitos básicos
Postura profissional
```
